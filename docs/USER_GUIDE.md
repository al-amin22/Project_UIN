# User Guide

## Untuk Admin

### Masuk ke Sistem

1. Buka halaman login.
2. Masukkan kredensial admin.
3. Masuk ke dashboard admin.

### Mengelola Akun

Admin dapat menambahkan data guru dan siswa melalui menu pengelolaan pengguna.

## Untuk Guru

### Melengkapi Profil

1. Masuk ke area guru.
2. Buka menu profil.
3. Isi data yang diperlukan.
4. Simpan profil.

### Menambahkan Materi

1. Buka menu materi.
2. Tambahkan materi baru.
3. Simpan konten pembelajaran.

### Menambahkan Latihan

1. Buka menu latihan.
2. Buat latihan baru.
3. Tambahkan soal pilihan ganda.
4. Simpan latihan.

### Menambahkan Kuis

1. Buka menu kuis.
2. Tambahkan soal kuis baru.
3. Isi pilihan jawaban dan kunci.
4. Simpan kuis.

### Mengelola Forum

1. Buka forum.
2. Tambahkan topik diskusi atau tanggapi pertanyaan siswa.

## Untuk Siswa

### Melengkapi Profil

1. Masuk ke area siswa.
2. Buka menu profil.
3. Isi data diri.
4. Simpan profil.

### Mengerjakan Latihan

1. Buka menu latihan.
2. Pilih latihan yang tersedia.
3. Jawab soal satu per satu.
4. Simpan jawaban.

### Mengerjakan Kuis

1. Buka menu kuis.
2. Klik mulai kuis.
3. Jawab setiap soal secara bertahap.
4. Simpan jawaban.
5. Selesaikan kuis setelah semua soal dijawab.

### Mengakses Materi

1. Buka menu materi.
2. Pilih materi yang ingin dipelajari.
3. Baca atau unduh e-book jika tersedia.

### Berpartisipasi dalam Forum

1. Buka halaman forum.
2. Pilih topik diskusi.
3. Kirim pertanyaan atau balasan.

## Tips Penggunaan

- pastikan role akun sesuai agar diarahkan ke dashboard yang benar
- gunakan latihan sebelum mengerjakan kuis
- baca materi dan petunjuk sebelum memulai aktivitas belajar
- simpan jawaban sebelum berpindah halaman bila sistem mengharuskan update bertahap

## Masalah Umum

### Tidak bisa login

- periksa email dan password
- pastikan akun sudah terdaftar

### Latihan atau kuis tidak muncul

- pastikan guru sudah membuat soal
- pastikan sesi belajar sudah aktif

### Jawaban tidak tersimpan

- isi semua field wajib
- pastikan koneksi stabil saat mengerjakan
